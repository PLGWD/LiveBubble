# LiveBubble

LiveBubble is a lightweight HTML overlay for OBS that lets you display animated speech bubbles while typing.

## Features

- 💬 Multiple speech bubbles
- ⌨️ Live typing preview
- ⏱️ Auto-hide after a configurable delay
- ✨ Smooth fade-out animation
- 🟢 Green background for OBS Chroma Key
- 📍 Bottom-left layout
- 🚀 No installation required

## Usage

1. Open `overlay.html`.
2. Open `control.html`.
3. Capture `overlay.html` in OBS.
4. Add a Chroma Key filter (green).
5. Type your message in `control.html` and press **Enter**.
